Highlights

1. Semester

Objektorientierte Programmierung

Ein Projekt, in dem wir eine eigene Interpretation des bekannten Spiels "Shufflepack Café" aus dem Jahre 1989 in Java programmieren durften.

2. Semester

Progr. Interaktiver Systeme

Ein Projekt, in dem wir eine Doppelverschlüsselung in Java programmieren durften. Das heisst, ein Text wird nacheinander mit zwei Codewörtern verschlüsselt.

2. Semester

Internetbasierte Systeme

Ein Projekt, in dem wir eine reine HTML Webseite programmieren sollen. Besonderes Augenmerk liegt auf strikt semantischem HTML und auf der Accessibility.
